# ruTorrent

ruTorrent是流行的Bittorrent客户端的前端 [rtorrent](http://rakshasa.github.io/rtorrent).


## 特点

* 轻量级-可以安装在小内存服务器或路由器上
* 扩展性-有可选插件，每个人都可以编写插件
* 画面优美
